本项目是基于SpringBoot2.0 + Spring Security构建的基础权限系统。

## 项目使用技术

* 服务端
	* Spring Boot2.0
	* Spring Security
	* Mybatis
	* MySQL8.0
	* slf4j
	* io.jsonwebtoken.jjwt
* 客户端
	* Vue.js
	* Vuetify
	* Material Design
